Criar Uma Página Web Utilizando as Tags Aprendidas

  • Estrutura Básica - < > e < / >
  • Título - <h1> à <h6>
  • Parágrafo - <p>
  • Destacar - <mark>
  • Diminuir - <small>
  • Itálico - <i>
  • Sublinhado - <u>
  • Ênfase - <strong>
  • Listas - <ol>, <ul> e <li>
  • Link - <a>
  • Linha - <hr>
  • Subscrito e sobrescrito - <sub> e <sup>
  • Citação - <blockquote>
  • Riscado -<del>
  • Abreviações - <abbr>

Estrutura Básica

  • Tags são os elementos básicos que estruturam e formatam um documento HTML. Elas indicam ao navegador como o conteúdo deve ser exibido ou interpretado.
  • <tag> - Tag de abertura
  • </tag> - Tag de fechamento

Título - <h1> à <h6>

  • Usadas para títulos e subtítulos;
  • <h1> é o mais importante (maior), <h6> o menos importante (menor);

Exemplo:

<h1>Titulo Principal</h1>
<h2>Subtítulo</h2>
<h3>Seção</h3>
<h6>Texto menor</h6>

Resultado:

Titulo Principal


Subtítulo


Seção


Texto menor

Parágrafo - <p>

  • Define um páragrafo de texto

Exemplo:

<p>Este é um parágrafo comum.</p>

Resultado:

Este é um parágrafo comum.

Destacar - <mark>

  • Destaca texto como se estivesse marcado com marca-texto.

Exemplo:

<p>Isso é um <mark> texto destacado </mark>. </p>

Resultado:

Isso é um texto destacado .

Diminuir - <small>

  • Deixa o texto menor, usado para observações.

Exemplo:

<p>Texto normal <small> texto destacado </small>. </p>

Resultado:

Texto normal texto destacado .

Itálico - <i>

  • Texto em itálico (ênfase leve).

Exemplo:

<p>Este texto está em <i> itálico </i> </p>

Resultado:

texto está em itálico

Sublinhado - <u>

  • Sublinhar texto.

Exemplo:

<p>Texto <u> sublinhado </u>. </p>

Resultado:

Texto sublinhado.

Ênfase - <strong>

  • Dá ênfase forte (negrito semântico)

Exemplo:

<p> <strong> Importante!</strong> Leia com atenção. </p>

Resultado:

Importante! Leia com atenção.

Listas - <ol>, <ul> e <li>

  • Lista ordenada (com números).
  • Lista não ordenada (com pontos).
  • Item da lista.

Exemplo:

<ol>
<li> Primeiro </li>
<li> Segundo </li>
</ol>

<ul>
<li> Item A </li>
<li> Item B </li>
</ul>

Resultado:

  1. Primeiro
  2. Segundo
  • Item A
  • Item B

Link - <a>

  • Cria um link.

Exemplo:

<a href="https://www.google.com"> Ir para o Google</a>

Resultado:

Ir para o Google

Linha - <hr>

  • Linha horizontal (separador)

Exemplo:

<hr>

Resultado:


Subscrito e sobrescrito - <sub> e <sup>

  • <sub> Texto subscrito (embaixo, usado em fórmulas químicas).
  • <sup> Texto sobrescrito (em cima, usado em potências).

Exemplo:

H<sub>2</sub>O
x<sup>2</sup>

Resultado:

H2O
x2

Citação - <blockquote>

  • Citação de um texto.

Exemplo:

<blockquote>"Este é um texto citado"</blockquote>

Resultado:

"Este é um texto citado"

Riscado -<del>

  • Mostra texto excluído (riscado).

Exemplo:

<p>Preço: <del> R$ 100,00 </del> R$ 80,00 </p>

Resultado:

Preço: R$ 100,00 R$ 80,00

Abreviações - <abbr>

  • Define abreviações com tooltip.

Exemplo:

<abbr title="HyperText Markup Language">HTML </abbr>

Resultado:

HTML
